보유한 기술 스택입니다

Java

  • Java 8+ 문법에 능숙하며, 함수형 프로그래밍 패러다임을 이해하고 활용할 수 있습니다.
  • 멀티스레딩과 동시성 프로그래밍에 대한 이해가 있으며, CompletableFuture를 활용한 비동기 프로그래밍 경험이 있습니다.
  • JUnit을 이용한 단위 테스트 작성에 익숙합니다.

Spring Boot

  • Spring Framework의 핵심 개념(IoC, DI, AOP)을 이해하고 있으며, 이를 활용한 애플리케이션 개발 경험이 있습니다.
  • Spring Data JPA를 이용한 데이터 접근 계층 구현에 능숙합니다.
  • Spring Security를 활용한 인증 및 권한 관리 구현 경험이 있습니다.
  • RESTful API 설계 및 구현에 능숙하며, Spring MVC를 활용한 웹 애플리케이션 개발 경험이 있습니다.

Docker

  • Dockerfile을 작성하여 애플리케이션을 컨테이너화할 수 있습니다.
  • Docker Compose를 사용하여 다중 컨테이너 애플리케이션을 정의하고 실행할 수 있습니다.
  • Docker 네트워크와 볼륨 관리에 대한 이해가 있습니다.

AWS

  • EC2를 사용한 가상 서버 생성 및 관리 경험이 있습니다.
  • RDS를 활용한 관계형 데이터베이스 운영 경험이 있습니다.
  • S3를 이용한 정적 파일 저장 및 관리 경험이 있습니다.
  • ELB를 통한 로드 밸런싱 구성 경험이 있습니다.
  • CloudWatch를 활용한 모니터링 및 로깅 경험이 있습니다.
